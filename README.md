# 🌤️ Weather Forecast App

A modern and responsive weather application that provides real-time weather information along with hourly and multi-day forecasts. The app automatically adapts to day and night themes based on the current time, delivering a seamless experience across desktop and mobile devices.

## 🔗 Live Demo

**Live Website:** [Click here](https://weather-forecast-app-adnankhan.netlify.app/)

## ✨ Features

- 📍 Search weather by city name
- 🌡️ Displays current weather conditions
- 🕒 24-hour weather outlook
- 📅 4-day weather forecast
- 💧 Shows humidity
- 🌬️ Displays wind speed
- 👁️ Shows visibility
- 🌙 Automatic Day/Night theme switching
- 📱 Fully responsive for mobile and desktop
- ⚠️ Custom error screen for invalid locations or failed API requests
- ⚡ Clean and user-friendly interface

## 🔧 How It Works

The application uses **two APIs**:

1. **Geocoding API**
   - Converts the entered city name into latitude and longitude coordinates.

2. **Weather API**
   - Uses the latitude and longitude to fetch:
     - Current weather
     - Next 24-hour forecast
     - 4-day weather forecast
     - Humidity
     - Wind speed
     - Visibility

This two-step approach ensures accurate weather information for every searched location.
